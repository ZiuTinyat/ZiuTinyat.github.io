Description

"Alice's Adventure" is a branch project of " Alice " held by ETC. This project aims to develop an adventure game creation tool that can be used to teach basic programming concepts as well as game design. Users are supposed to use drag-drop objects and block-based coding to create simple 2D adventure games with individualized mechanics.

The application is written in HTML5 and Javascript, using Electron and Node.js to generate PC/macOS versions. The game engine is based on Pixi.js and will be encapsulated into adventure game specified "Alice Engine". Blockly is also customized and implanted into this project to provide block-based coding interface. The application will have functions of save/load projects, playtest in editor, compile project, etc.

My contribution

As a Lead Programmer, my first job is to determine our platforms and tools. I concluded tons of researches we did about languages and tools including C++ (Cocos), C# (Unity), Java (BladeCoder) and JS (Pixi, Electron). It turned out that JS best serves our needs that it is higher efficiency than C++ and suitable for fast prototyping; it provides more flexible UI design than Unity; it is easy to produce cross-platform products; it has tons of free libraries online that we can take advantages of.

After deciding our language, I learnt to use our major tools and did a gold spike application to connect Electron, Pixi and Blockly together.

I also designed the system architecture.

As a Front-end Programmer, I implemented all front-end logics including UI-data binding, digitalizing project data, saving/loading project files, running playtest and so on. I utilized require.js to modularize my code and encapsulated all classes into global AliceEditor object.
