Introduction to OpenPose

"OpenPose" is a most popular open-source library on GitHub by CMU Perceptual Computing Lab. More details can be found from this link: OpenPose Information.

Working with CMU Perceptual Computing Lab, I developed an Unity demonstration for OpenPose on CVPR 2018. After that, I have been working on developing an OpenPose plugin for Unity Users. (Link to GitHub repository)

OpenPose Unity demo

This project utilizes OpenPose multi-camera 3D reconstruction function, and renders the output in Unity in real time. This demonstration was intended to be used in CVPR 2018. In Unity, I implemented skeletal animating system, file parsing system and frame control system. And in C++, I implemented data formating and UDP streaming functions.

The real-time 3D reconstruction works as:

  1. OpenPose sends JSON formated output data through UDP.
  2. Unity receives and stores the data.
  3. The frame control system control the flow of data to ensure the smoothness.
  4. The animating system converts the data and plays the animation

The UDP streaming system was implemented on both side (C++ library and Unity). In C++ side, This demonstration project was successfully done within 1 month, with complete GUI system and full documentation.

OpenPose Plugin

This project aims to integrate OpenPose C++ plugin into Unity, and publish a free package in Unity Assets Store.
