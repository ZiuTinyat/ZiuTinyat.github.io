Introduction to OpenPose

"OpenPose" is a most popular open-source library on GitHub by CMU Perceptual Computing Lab. More details can be found from this link: OpenPose Information.

Working with CMU Perceptual Computing Lab, I developed an Unity demonstration for OpenPose on CVPR 2018. After that, I have been working on developing an OpenPose plugin for Unity Users. The GitHub repository can be accessed by this link.
