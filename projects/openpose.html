OpenPose

"OpenPose" is a popular open-source CV project on GitHub comducted by CMU Perceptual Computing Lab. It supports body, hand, face keypoints recognition for single image, video input and camera (live). It also supports 3D reconstruction (experimental only) using more than 2 cameras. More details can be found from this link: OpenPose Information.
